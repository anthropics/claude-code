# Performance Optimizer Plugin

Analyzes code and recommends cloud resource optimization, algorithmic improvements, and infrastructure optimization to save money and improve performance.

## Overview

Performance Optimizer analyzes your code and infrastructure to identify optimization opportunities with real cost/performance impact. It estimates carbon footprint reduction, cost savings, and latency improvements for each optimization.

## Features

- **Algorithmic Optimization**: Suggests more efficient algorithms with complexity analysis
- **Cloud Cost Analysis**: Identifies expensive usage patterns and recommends cheaper alternatives
- **Database Query Optimization**: Analyzes slow queries and suggests indexes/rewrites
- **Infrastructure Right-Sizing**: Recommends appropriate server sizes and configurations
- **Caching Strategy**: Identifies caching opportunities and recommended solutions
- **Cost-Benefit Analysis**: Every optimization includes savings estimates
- **Carbon Tracking**: Estimates CO2 reduction from optimizations
- **Before/After Simulation**: Simulates performance improvements

## Command: `/optimize-performance`

Analyzes code for optimization opportunities.

**Usage:**
```bash
/optimize-performance
```

## Optimization Areas

- Algorithm complexity reduction
- Database query optimization
- Caching strategies
- Server/container right-sizing
- CDN optimization
- Database indexing
- API pagination
- Batch processing opportunities
- Asynchronous processing candidates
- Resource allocation efficiency

## Cost Estimation

For each optimization:
- Monthly cost savings estimate
- Implementation difficulty
- Risk assessment
- Payback period
- Carbon footprint reduction
