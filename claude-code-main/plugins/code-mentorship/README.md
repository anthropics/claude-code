# Code Mentorship Plugin

AI-driven mentorship engine that teaches junior developers best practices and formats code explanations in each developer's preferred learning style.

## Overview

Code Mentorship transforms how junior developers learn. Instead of just writing code, it teaches *why* decisions were made, shows *patterns* they can apply elsewhere, and adapts explanations to each developer's learning style—visual, step-by-step, conceptual, or practical examples.

## Features

- **Learning Style Adaptation**: Visual learners, visual-step-by-step, conceptual, practical
- **Interactive Code Explanations**: Explains not just what code does, but why it was written that way
- **Pattern Recognition**: Teaches generalizable patterns, not just solutions to current problems
- **Guided Learning**: Asks clarifying questions to ensure understanding
- **Concept Building**: Builds knowledge progressively from fundamentals
- **Real-World Context**: Connects code to business problems
- **Practice Assignments**: Generates homework to reinforce learning
- **Progress Tracking**: Monitors learning and adapts explanations

## Command: `/mentor-explain`

AI mentors explain code and teach concepts.

**Usage:**
```bash
/mentor-explain This React component that fetches user data
```

## Learning Styles Supported

- **Visual**: Diagrams, flowcharts, component trees
- **Visual-Sequential**: Step-by-step visual walkthrough
- **Conceptual**: High-level ideas before code details
- **Practical**: Real-world examples, use cases, anti-patterns
- **Socratic**: Questions guiding to the answer
- **Comparative**: Compare against wrong approaches

## Teaching Domains

- Frontend: React, Vue, Angular, HTML/CSS
- Backend: Node.js, Python, Go, Java
- Databases: SQL, MongoDB, RESTful design
- DevOps: Docker, Kubernetes, deployment
- Architecture: Microservices, design patterns, scalability
- Testing: Unit, integration, end-to-end testing
- Performance: Optimization, profiling, benchmarking
