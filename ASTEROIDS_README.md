# Terminal Asteroids Game

A fun asteroids-style game you can play right in your terminal!

## How to Play

Run the game with:
```bash
python3 terminal_asteroids.py
```

## Controls

- **Arrow Keys**: Rotate and move your ship
  - ⬅️ Left Arrow: Rotate counterclockwise
  - ➡️ Right Arrow: Rotate clockwise
  - ⬆️ Up Arrow: Thrust forward
- **Space**: Shoot bullets
- **Q**: Quit game
- **R**: Restart (when game over)

## Gameplay

- Destroy all asteroids to advance to the next level
- Large asteroids split into medium ones, medium split into small ones
- Avoid colliding with asteroids or it's game over!
- Your ship has momentum - use thrust carefully
- Bullets are limited to 5 on screen at once

## Scoring

- Small asteroid: 30 points
- Medium asteroid: 20 points
- Large asteroid: 10 points

Have fun and see how high you can score!
