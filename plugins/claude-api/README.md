# Claude API Plugin

A skill for building LLM-powered applications using the Claude API and official Anthropic SDKs.

## Skills

| Skill | Description |
|-------|-------------|
| [claude-api](./skills/claude-api/) | Complete guide to the Anthropic API — basic usage, streaming, tool use, vision, and best practices |

## Usage

The skill is auto-invoked when you ask Claude to help build with the Claude API:

- "How do I use the Anthropic SDK in Python?"
- "Build a chatbot using Claude"
- "Add tool use to my Claude integration"
- "Help me stream Claude responses in TypeScript"

## Covered Topics

- SDK installation and authentication
- Basic message creation (Python & TypeScript)
- Model selection guide
- Streaming responses
- Multi-turn conversations
- Tool use / function calling
- Vision (image input)
- Error handling and best practices

## Installation

Include this plugin in your Claude Code configuration to enable Claude API development guidance.
