# security-guidance

Security reminder hook that warns about potential security issues when editing files, including command injection, XSS, and unsafe code patterns.

## Description

This plugin provides hooks that help identify potential security vulnerabilities in code as you work with files in Claude Code.

## Features

- Warns about command injection vulnerabilities
- Detects potential XSS issues
- Identifies unsafe code patterns
- Proactive security guidance during development
