# Remember

Save to Snipara memory for future recall.

Use `rlm_remember` with:
- `content`: "$ARGUMENTS"
- `type`: Infer from content (fact, decision, learning, preference, todo, context)
- `scope`: "project" (default)

**Types:**
- **decision** - "We chose X because Y"
- **learning** - "Pattern X works well for Y"
- **context** - "Completed X, next: Y"
- **fact** - "API uses JWT auth"
- **todo** - "Need to implement X"
- **preference** - "User prefers dark mode"
